The Mission Behind Lunar Observatory

Lunar Observatory is an interactive astronomy platform designed to make celestial mechanics intuitive, visual, and accessible for everyone.

The Idea

Astronomy is often taught through equations and static diagrams, but the universe itself is dynamic. Planets move, phases change, and celestial events unfold in time. Lunar Observatory was created to make these motions visible and intuitive.

Instead of reading about orbital mechanics in a textbook, users can observe it directly. By combining real astronomical calculations with modern 3D graphics, the project allows anyone to explore the solar system as a living system rather than a collection of numbers.

The goal is simple: turn complex astronomical data into an experience that is visual, interactive, and educational.

Technology

Three.js

A powerful WebGL rendering engine used to simulate the solar system in real time. It enables smooth camera movement, dynamic lighting, and interactive exploration.

Swiss Ephemeris

A professional astronomical library used for high-precision planetary calculations. It provides accurate ephemeris data to reproduce real celestial positions.

Python Backend

Scientific computations such as planetary positions, event prediction, and visibility calculations are handled by a Python backend optimized for data processing.

Capacitor

The web application is wrapped into a native Android app using Capacitor, allowing integration with mobile features such as offline data storage.

Scientific Accuracy

Lunar Observatory aims to represent celestial motion as faithfully as possible. Planetary positions, lunar phases, and astronomical events are calculated using astronomical models and verified ephemeris data.

While the application is designed primarily for education and exploration, the underlying calculations follow real astronomical conventions. This ensures that the simulated positions and phases closely match the actual sky.

The goal is not only to visualize the universe but also to encourage curiosity about the physics and mathematics that govern it.
