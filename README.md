# 🌙 Lunar Observatory

An interactive astronomy platform for exploring celestial mechanics, planetary motion, and astronomical events.

Built as a physics-driven visualization project combining **Three.js, WebGL, and modern web technologies**.

---

## 🚀 Platforms

Lunar Observatory is available in three forms:

| Platform | Description |
|--------|-------------|
| 🌐 PWA | Browser version hosted on GitHub Pages |
| 📱 Android | Native APK built using Capacitor |
| 💻 Desktop | Planned offline desktop version |

---

## ✨ Features

- Real-time **solar system visualization**
- Accurate **moon phase simulation**
- **Planet visibility calculations**
- **Astronomical event predictions**
- Interactive **time travel mode**
